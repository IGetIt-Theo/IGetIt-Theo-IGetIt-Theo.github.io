US Baby Names Analysis (1921-2021)

Power BI | DAX | Google BigQuery | January 2025

Overview

Interactive dashboard analyzing 100 years of US baby name trends using SSA data. The visualization reveals fascinating cultural shifts, including how names like "Charlie" transitioned from predominantly male (90%+ in the 1920s) to predominantly female (60%+ by 2021).

Key Features

  • Searchable name database - Type partial names (e.g., "Charl") to explore variations
  • Historical rank tracking - Shows peak popularity and year achieved
  • Gender distribution analysis - Visualizes how naming patterns shift over time
  • Geographic heat mapping - Per-capita popularity by state (continental US)

Technical Approach

Data modeling: Structured star schema with dimension tables for names, dates, and states, connected to a fact table of birth records.

Data filtering: Limited to names with 1,500+ total births to focus on culturally significant trends and improve performance.

Key calculations:

  • DAX RANKX function for historical rank calculations across 100+ years
  • Per-capita popularity (births with name ÷ total births by state)
  • Gender percentage trends over time

Tools: Power BI Desktop, DAX, Google BigQuery, Excel Power Query

Insights

The dashboard reveals several compelling patterns:

  • Names like Charlie, Taylor, and Ashley show dramatic gender shifts over decades
  • Regional naming preferences persist even as national trends change
  • Popularity peaks are often sharp and short-lived (e.g., Jennifer in the 1970s-80s)
